## 📦 Что это?

**Setup Modern Desktop.exe** — это SFX-архив (самораспаковывающийся архив WinRAR), который содержит полный набор инструментов для автоматической кастомизации Windows 10.

После запуска архив:
1. Распаковывается в `C:\Modern`
2. Автоматически запускает `start.bat`
3. Устанавливает все компоненты без лишних вопросов

---

## 🚀 Установка

### 1. Скачайте файл
```
Setup Modern Desktop.exe
```

### 2. Запустите **ТОЛЬКО ОТ ИМЕНИ АДМИНИСТРАТОРА**

```batch
Правый клик на файле → "Запуск от имени администратора"
```

**ИЛИ через командную строку:**
```batch
runas /user:Administrator "Setup Modern Desktop.exe"
```

### 3. Дождитесь завершения

Установка занимает **3-5 минут**. Ничего не нажимайте — процесс полностью автоматический.

### 4. Перезагрузитесь

После завершения скрипт предложит перезагрузить компьютер.

## ⚡ Что установится автоматически?

| Компонент | Что делает |
|-----------|------------|
| **UltraUXThemePatcher** | Патчит систему для использования сторонних тем |
| **Sweetness Theme** | macOS-подобная тема оформления |
| **KUDA 5.0 White** | Белые минималистичные курсоры |
| **material_design_dark** | Альтернативные темные курсоры |
| **StartIsBack** | Кастомное меню Пуск |
| **Winstep Nexus** | Док-панель как на macOS |
| **OldNewExplorer** | Классический вид Проводника |
| **Кастомные обои** | Красивые обои рабочего стола |
| **Настройки реестра** | Цвета, прозрачность, поведение |
