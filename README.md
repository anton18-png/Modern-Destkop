## 📦 Что это?

**Setup Modern Desktop.exe** — это SFX-архив (самораспаковывающийся архив WinRAR), который содержит полный набор инструментов для автоматической кастомизации Windows 10.

После запуска архив:
1. Распаковывается в `C:\Modern`
2. Автоматически запускает `start.bat`
3. Устанавливает все компоненты без лишних вопросов

---

## 🚀 Установка

### 1. Скачайте файл
```
Setup Modern Desktop.exe
```

### 2. Запустите **ТОЛЬКО ОТ ИМЕНИ АДМИНИСТРАТОРА**

```batch
Правый клик на файле → "Запуск от имени администратора"
```

**ИЛИ через командную строку:**
```batch
runas /user:Administrator "Setup Modern Desktop.exe"
```

### 3. Дождитесь завершения

Установка занимает **3-5 минут**. Ничего не нажимайте — процесс полностью автоматический.

### 4. Перезагрузитесь

После завершения скрипт предложит перезагрузить компьютер.

## 📋 Шаги установки (что делает скрипт)

### 1. ПРОВЕРКА ULTRAUXTHEMEPATCHER
- **ULTRAUXTHEMEPATCHER ДОЛЖЕН БЫТЬ УСТАНОВЛЕН**
- Проверяется наличие папки `C:\Program Files (x86)\UltraUXThemePatcher`
- Если не установлен — запускается `UltraUXThemePatcher_4.4.4.exe`

### 2. ОПРЕДЕЛЕНИЕ ПЕРЕМЕННЫХ
- Устанавливаются пути для всех компонентов
- Определяется SID текущего пользователя

### 3. УСТАНОВКА WINSTEP NEXUS
- Тихая установка `NexusSetup.exe`
- Принудительное завершение процесса Nexus

### 4. КОПИРОВАНИЕ WINSTEP
- Копирование папки `Winstep` в `C:\Users\Public\Documents\Winstep`
- Импорт `Winstep.reg` в реестр
- Запуск Nexus

### 5. КОПИРОВАНИЕ ЗАДАНИЯ ПЛАНИРОВЩИКА
- Копирование `StartIsBack health check` в `C:\Windows\System32\Tasks`

### 6. КОПИРОВАНИЕ ТЕМЫ SWEETNESS
- Копирование папки `Sweetness` в `C:\Windows\Resources\Themes\Sweetness`

### 7. КОПИРОВАНИЕ КУРСОРОВ
- `material_design_dark` → `C:\Windows\Cursors\material_design_dark`
- `KUDA 5.0 White` → `C:\Windows\Cursors\KUDA 5.0 White`
- Применение настроек курсоров в реестре

### 8. УСТАНОВКА STARTISBACK
- Копирование `StartIsBack` в `%ProgramFiles(x86)%\StartIsBack`

### 9. КОПИРОВАНИЕ ORBS (КНОПКА ПУСК)
- Копирование `Orbs` в `%LOCALAPPDATA%\StartIsBack\Orbs`

### 10. КОПИРОВАНИЕ ТЕМ ПОЛЬЗОВАТЕЛЯ
- Копирование `Themes` в `%LOCALAPPDATA%\Microsoft\Windows\Themes`

### 11. УСТАНОВКА ОБОЕВ
- Копирование `Wallpapers\4.jpg` в профиль пользователя
- Применение через реестр

### 12. ПРИМЕНЕНИЕ НАСТРОЕК FIX
- Системные настройки (Input Locales)
- Цвета акцента (`FFD77800` и `FF9E5A00`)
- Настройки панели задач и проводника
- **Панель задач СВЕРХУ** (импорт `taskbar_top.reg` и `fix.reg`)

### 13. НАСТРОЙКА STARTISBACK
- Более 30 параметров реестра для кастомизации меню Пуск

### 14. НАСТРОЙКА ПОИСКА
- Обновление `InstalledWin32AppsRevision`

### 15. ПРИМЕНЕНИЕ ТЕМЫ SWEETNESS
- Установка текущей темы
- Настройка ThemeManager

### 16. НАСТРОЙКА ЦВЕТОВ DWM
- Цвет акцента
- Цвета Colorization

### 17. СКРЫТИЕ ИКОНОК РАБОЧЕГО СТОЛА
- Скрытие "Корзины" и "Этот компьютер"

### 18. ЗАМЕНА СИСТЕМНЫХ DLL
- Создание бэкапов:
  - `imageres.dll.bak`
  - `imagesp1.dll.bak`
  - `zipfldr.dll.bak`
- Замена DLL из папки `Discord\` через NSudoLG
- Очистка кэша иконок

### 19. НАСТРОЙКА OLDNEWEXPLORER
- Отключение капшона
- Отключение иконок
- Отключение ленты
- Отключение кнопки "Вверх"

### 20. РЕГИСТРАЦИЯ STARTISBACK
- `StartIsBack.exe /S`
- Повторный импорт `fix.reg`

### 21. ПЕРЕЗАПУСК ПРОВОДНИКА
- Принудительное завершение `explorer.exe`
- Запуск заново
- Обновление параметров системы

### 22. ПЕРЕЗАГРУЗКА
- Предложение перезагрузить компьютер
- При подтверждении — немедленная перезагрузка

## 🔄 После перезагрузки

1. Панель задач должна быть **СВЕРХУ**
2. Док Nexus должен быть **ВНИЗУ**
3. Тема Sweetness должна быть **АКТИВНА**
4. Курсоры KUDA 5.0 White должны быть **ПРИМЕНЕНЫ**
5. Иконки "Корзина" и "Этот компьютер" должны быть **СКРЫТЫ**
